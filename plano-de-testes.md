# Plano de Testes – SauceDemo

## 1. Objetivo
Garantir a qualidade das principais funcionalidades do site SauceDemo, validando se o sistema atende aos requisitos esperados por meio de testes manuais.

## 2. Escopo
Serão testadas as seguintes funcionalidades:
- Login do usuário
- Listagem de produtos
- Carrinho de compras
- Checkout
- Logout

## 3. Fora de escopo
- Testes de performance
- Testes de segurança
- Testes de compatibilidade entre navegadores

## 4. Ambiente de testes
- Sistema operacional: Windows
- Navegador: Google Chrome
- URL: https://www.saucedemo.com/

## 5. Massa de dados
Usuários fornecidos pelo próprio site:
- standard_user
- locked_out_user
- problem_user

## 6. Critérios de entrada
- Sistema disponível
- Ambiente configurado

## 7. Critérios de saída
- Casos de teste executados
- Bugs registrados
- Evidências coletadas
