# Projeto QA Júnior – SauceDemo

Projeto prático de QA Júnior com foco em testes manuais, documentação de testes e registro de bugs, utilizando o site SauceDemo.

## Objetivo
Demonstrar conhecimentos iniciais em Qualidade de Software por meio de planejamento de testes, criação de casos de teste e documentação.

## Ferramentas
- Testes manuais
- Documentação em Markdown
- Navegador Web

## Sistema testado
https://www.saucedemo.com/
